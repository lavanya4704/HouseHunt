#  HouseHunt
HouseHunt is a full-stack MERN (MongoDB, Express.js, React.js, and Node.js) real estate rental platform that simplifies property discovery and management. The application enables users to securely register and log in, browse and search verified property listings, view detailed property information with Google Maps integration, save properties to a wishlist, submit reviews and ratings, schedule property visits, and contact property owners via Email or WhatsApp. Property owners can manage listings, monitor visit requests, and accept or reject appointments through a personalized dashboard. The project demonstrates secure JWT authentication, RESTful APIs, responsive UI design, and scalable full-stack web development.
